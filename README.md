Project Overview

This project involves designing and analyzing a relational Bookstore Database using PostgreSQL. The database models real-world bookstore operations including books, customers, orders, and sales transactions.

The objective of this project is to perform SQL-based data analysis to extract meaningful business insights from structured data.

Tools Used
• PostgreSQL
• pgAdmin
• GitHub

 Database Schema
 The project includes the following tables:
• books
• authors
• customers
• orders
• order_details

Relationships were created using primary and foreign keys to maintain referential integrity.

 SQL Concepts Used
 This project demonstrates:
• SELECT statements
• WHERE clause
• GROUP BY
• HAVING
• ORDER BY
• Aggregate functions (SUM, COUNT, AVG, MAX)
• INNER JOIN
• LEFT JOIN
• Subqueries

 Analysis Performed
• Total number of books available
• Best-selling books
• Total revenue generated
• Top customers by purchase amount
• Monthly sales analysis
• Average order value
